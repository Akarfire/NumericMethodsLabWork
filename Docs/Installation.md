
All commands must be run inside of the `Electron` subfolder.

### Node JS libraries install commands

`npm install --save-dev electron`


### Python libraries install commands

`./python/python.exe ./python/get-pip.py`

`./python/python.exe -m pip install numpy`

`./python/python.exe -m pip install scipy`

`./python/python.exe -m pip install plotly`
