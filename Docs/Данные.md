
# Данные

### Вввод

1. Основные
	1. Кол-во партий Свёклы для переработки (`int n`);
	2. Масса одной партии (`float`);
	3. Диапазон начальной сахаристости партии (`range float A`);
	4. Диапазон деградации (`range float B`);
	5. Режим деградации: равномерный и концентрированный;
		1. Для концентрированного размер концентрированного;
		2. *Если можно: сторону сжатия;*
	6. Кол-во экспериментов;
	7. *Если можно: диапазоны для каждой партии;*

2. Неорганические в-ва (Для каждой партии отдельно):
	1. Кальций (`range float`);
	2. Натрий (`range float`);
	3. Азот (`range float`);
	4. Начальное содержание редуцирующих в-в (`range float`);

3. Дозаривание:
	1. Кол-во этапов дозаривания (`int`);
	2. Диапазон дозаривания (`range float`, > 1);

### Вывод

1. График;
	1. Выбор отдельных стратегий;
	2. КАСИВО!!!

2. То, что на графике, но цифрами;

3. Выводы:
	1. Лучшая стратегия;
	2. Худшая стратегия;
	3. Отрезок;
